# SnowRunner Truck Color Explorer and Editor

This is mashup of a few projects over the span of a few years,
and which haven't been fully unified yet,
but it's complete.

## Parser

The `./parser/` directory contains a NodeJS app to parse the game file
into two JavaScript files: one with just all the truck names and the
other with all the truck color information. The parser does a couple
more things. It converts the color information into hex and SnowRunner's
color format (for instance, `g(255; 0; 127)` which can be useful if you
are creating a mod); it also generates five color harmonies: analogous,
triadic, square, complementary, and split complementary.

## app

The `./app/index.html` file uses the JavaScript files generated by the parser
to create a visualization. The "Color Explorer" tab presents all the
truck colors from the game. Clicking one of the grid items will open
the colors in the "Color Editor" tab, which works like the in-game color
editor.

In the editor tab, clicking on one of the color
harmonies will copy the color to the clipboard and you can right click
one of the editable colors to paste it and update the sliders. Use the
sliders or the number input fields to adjust the values.
The "Reset Colors" tab resets to the default cyan color of the in-game
color editor.

Selecting the explorer tab will take you back to where you left off.
